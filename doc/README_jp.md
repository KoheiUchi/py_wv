# py_wv.py

*[English README](README.md)*

ngspice の **バイナリ `.raw`** 波形を読み込み、表示・重ね合わせ・上下分離・
2点間測定ができる **単一ファイル** の波形ビューアです。Python + pyqtgraph
(PyQt6) 製。

`py_wv.py` は `waveview` パッケージ（rawfile リーダ・測定処理・Qt GUI）を1つの
スクリプトに統合したものです。隣に `waveview/` パッケージを置く必要はなく、
`numpy` と `pyqtgraph` があれば単体で動作します。

## 起動

```bash
python3 source/py_wv.py                 # 空の状態で起動 → メニューから読み込み
python3 source/py_wv.py outputfile.raw  # ファイルを指定して起動
# 実行権限があれば
chmod +x source/py_wv.py
./source/py_wv.py outputfile.raw
```

依存: `numpy`, `pyqtgraph`, Qt バインディング（`PyQt6` を使用）。
`pip install numpy pyqtgraph PyQt6` でインストールできます。

> UI は既定で **英語** で起動します。**Option → Language** でいつでも日本語に切替可能です。

## 機能と操作

### ファイル読み込み
- ツールバー **Open .raw…** から複数ファイルをまとめて選択可。
- **同名ファイルも何度でも読み込み可能**。各読み込みに一意の `#番号` が付与され、
  `Sources` ツリーや測定対象で区別されます（例: `outputfile.raw #1`, `#2`）。

### ファイルメニュー
- **Open .raw…（Ctrl+O）** … `.raw` を読み込む（複数可）。
- **Close .raw（Ctrl+W）** … **Sources で選択した `.raw`** を閉じる（アンロード）。
  ファイルノード／その信号のどちらを選んでいても、そのファイルが閉じられ、
  関連するトレース・測定対象も自動的に取り除かれます。複数選択も可。
- **Exit（Ctrl+Q）** … プログラムを終了。

### 波形の表示
- 左の **Sources** ツリーで信号のチェックボックスを ON にすると表示トレースに追加。
- `time`（掃引軸）は自動的に X 軸になります。
- **X 軸の目盛は SI 接頭辞（10³刻み: …, ps, ns, µs, ms, …）+単位**で表示されます
  （例: `60ns`）。上下に分離した全ペインで同じ書式になり、表示範囲に応じて
  接頭辞は自動で切り替わります。

### 重ね合わせ / 上下分離
- 各トレースには **Pane（ペイン番号）** があります（左の **Traces** 表）。
  - **同じ Pane 番号 = 重ねて表示**。
  - **異なる Pane 番号 = 上下に分離表示**（X 軸は全ペイン連動）。
- 表でトレースを選択して：
  - ツールバー **Split selected ▼** … 選択を新規ペインへ分離。
  - ツールバー **Merge selected ▲** … 選択を1つのペインへ重ねる。
  - Pane 列のスピンボックスで個別に番号変更も可能。
- 表の列幅は変更可能です。**Trace 列の境界をドラッグ**すると広げられます。

### 表示・ズーム（View メニュー）
- **Fit All（全体表示, Ctrl+0）** … 全ペインを全データが収まる範囲に戻す。
- **Zoom In / Zoom Out（拡大 / 縮小, Ctrl++ / Ctrl+-）** … 中心基準で拡大縮小。
  X 軸は全ペイン連動、Y 軸は各ペイン個別。
- **Rectangle Zoom（範囲選択ズーム, Ctrl+R）** … ON にすると、プロット上を
  **左ドラッグして任意の2点で囲んだ矩形範囲にズーム**。OFF でパン操作に戻る。
- **Zoom X to Cursors A–B（カーソル間にズーム, Ctrl+B）** … 測定カーソル A–B の
  区間に X 範囲を合わせる。
- **Panels（パネル表示）** … Sources / Traces / Measure の各ドックの表示/非表示。
  ✕ で閉じたパネルはここから再表示できます。
- **X 軸ナビゲーションバー** … プロットの下にあるボタン列で、X 軸（全ペイン連動）を
  マウスで操作できます（キーボード操作と同等）:
  - **◀◀ / ◀** … 左へ移動（表示範囲幅の 50% / 10%）。
  - **－ / ＋** … 表示中心を基準に X 軸を縮小 / 拡大（Y 軸は変化しません）。
  - **▶ / ▶▶** … 右へ移動（10% / 50%）。
- **矢印キーで横スクロール** … プロットをクリックしてフォーカスを当てた状態で、
  **← / →** で波形を左右に移動（表示範囲幅の 10%）。**Shift+← / →** で大きく移動（50%）。
  （プロットにフォーカスがある時のみ動作するため、スピンボックスやツリーの
  矢印操作とは干渉しません。）
- **軸ごとのマウス操作**（メニュー不要・常時有効）:
  - **X 軸の上**でドラッグ / ホイール → **X 軸だけ**ズーム・移動。
  - **Y 軸の上**でドラッグ / ホイール → **Y 軸だけ**ズーム・移動。
  - プロット内のホイール → 両軸ズーム、右ドラッグ → ズーム、左ドラッグ → パン
    （Rectangle Zoom が OFF のとき）。

### オプション（Option メニュー）
- **Language（言語）** … メニューやラベルの表記を **English / 日本語** で切替（即時反映）。
  起動時の既定は英語です。
- **Font（フォント）** … UI の表示フォントサイズを変更。
  - **Enlarge（拡大, Ctrl+Shift+.）**
  - **Shrink（縮小, Ctrl+Shift+,）**
  - **Default（デフォルトに戻す, Ctrl+Shift+0）**
  - （`=` `+` `-` は日本語キーボードでは Shift 合成のためショートカットとして
    機能しにくく、`.` `,` `0` を採用しています。）

### 線の色・パターン
- **Traces** 表の **Color** ボタンで色を選択。
- **Style** で `Solid / Dash / Dot / DashDot / DashDotDot` を選択。

### 2点間測定
右の **Measure** パネルで Point A / Point B を設定します。
- **Trace**: 測定対象の波形（ファイル＋信号）。A と B で別波形を指定可。
- **Ref**（基準）:
  - `Cursor` … カーソル X 位置でのトレース値。
  - `Level` … 指定レベルの交差点（立上り/立下り問わず、表示範囲内）。
  - `Rise`  … 立上りエッジでのレベル交差（表示範囲内）。
  - `Fall`  … 立下りエッジでのレベル交差（表示範囲内）。
  - `Level` / `Rise` / `Fall` は **現在表示中の X 範囲内** の交差点のみを対象とし、
    複数該当する場合は **一番右（X 最大）** の点を採用します。パン／ズームで
    表示範囲を変えると Point は自動的に選び直されます。
- **Level**: 交差判定レベル。**½·max** ボタンで「選択トレース最大値の 1/2」を自動設定
  （例: Vdd=5V → 2.5V）。
- プロット上の **縦カーソル A（黄）/ B（水）をドラッグ**すると、`Cursor` では
  カーソル位置に、`Rise`/`Fall`/`Level` では表示範囲内で一番右の交差点にスナップし、
  マーカー（●）が表示されます。
- 結果に **Δt**, **1/Δt**, **ΔV / ΔI**（両点の量に応じて自動切替）を表示。
  交差はサンプル間を線形補間するため、シミュレーション刻みより細かい分解能が出ます。

## 構成

`py_wv.py` は元のパッケージ構成に対応した3つのセクションからなる単一スクリプトです。

```
py_wv.py
  ├─ rawfile   # バイナリ .raw パーサ + データモデル（real/complex 対応）
  ├─ measure   # エッジ/レベル交差検出・補間
  └─ app       # GUI 本体（pyqtgraph）+ main() エントリポイント
```
